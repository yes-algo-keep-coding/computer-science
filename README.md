## :scroll: Rules

###   :round_pushpin: Commits

- #### Title: {Commit_Type}: {Subject}

- **Title은 50자 미만, Body는 부가 설명을 간략하게 기재**

- **Commit Types**

  - **Docs**: 새로운 문서의 추가 및 수정
  - **Refactor**: 기능 변화가 없는 버그 수정 및 코드의 변경
  - **Style**: 코드 의미에 영향을 주지 않는 변경 사항
  - **Correct**: 문법의 오류, 오타 등의 정정
  - **Chore**: 기타 변경 사항
  - **Revert**: 작업 되돌리기
